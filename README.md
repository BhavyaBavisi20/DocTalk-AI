# DocTalk AI – Document Q&A Assistant

DocTalk AI is a **GenAI-powered document question-answering assistant** designed to retrieve, understand, and summarize **medical documents** using **Retrieval-Augmented Generation (RAG)**.  
The system enables fast, accurate, and context-aware conversations over large document collections.

---

## Key Features
-  **Medical Document Q&A** using Retrieval-Augmented Generation (RAG)
-  **Context-aware conversations** powered by LangChain
-  **Semantic search** with FAISS vector database
-  Faster document retrieval and improved response accuracy
-  **Text-to-Speech support** using Google Text-to-Speech
-  Interactive chat interface built with Streamlit

---

## Tech Stack
- **Python**
- **Streamlit** – UI and chatbot interface
- **LangChain** – Orchestration and conversational retrieval
- **FAISS** – Vector database for semantic search
- **Google Text-to-Speech** – Audio responses
- **LLMs / GenAI models**
- **RAG (Retrieval-Augmented Generation)**
